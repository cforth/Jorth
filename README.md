Jorth -- Forth system by Java
====================

###### Author:Chai Fei
###### E-mail:cforth@cfxyz.com
--------------------

Jorth是一门用Java实现的Forth方言！！！

1. 实现了整型数据类型；
2. 实现了常量、变量、数组三种数据结构的读取与写入；
3. 实现了条件分支、循环两种控制语句；
4. 实现了扩展词（函数）的定义与执行；

Jorth语言可以实现死循环（没有循环或递归受限）， 可以实现数组这样的数据结构(能模拟纸带)。所以Jorth语言能模拟出图灵机，完成了图灵完备。
